## Requirement
* 自分のブラウザに合ったバージョンのWebDriver  
* Selenium 4.14.0
* Flet

## Installation
For Windows Users:
Under ReUNIPA directory
```
python3 -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
cd src
flet pack main.py
```
